Comandos Basicos do sistema Operacional Windows(Terminal)

dir : traz a lista de diretório dentro da pasta
cd / : navegar entre as pastas
cd .. : retroceder um passo anterior
cls : limpar o terminal do windows
tab : auto completar (atalho do teclado)
mkdir : criar uma pasta
del : deletar arquivos
rmdir : deleta a pasta e todos os seus arquivos


Comandos Basicos do sistema Operacional Linux(Terminal)

ls : traz a lista de diretório dentro da pasta
cd / : navegar entre as pastas
cd .. : retroceder um passo anterior
clear : limpar o terminal ou atalho no teclado ( ctrl + L )
tab : auto completar (atalho do teclado)
mkdir : criar uma pasta
rm : deletar arquivos na pasta
rm -rf : deleta a pasta e todos os seus arquivos
